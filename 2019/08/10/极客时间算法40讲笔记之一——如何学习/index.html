如何有效学习数据结构

  1. Chunk it up (切碎知识点)
    比如要学习算法与数据结构，我们可以把想要学习的数据结构和算法罗列出来，在整理出他们之间的关系，然后各个击破。

  1. Deliberate practicing (刻意练习)
    • 练习缺陷、弱点的地方
    • 练习时会不舒服、不爽、枯燥
    • 会有一个下降后上升的曲线，重复这个曲线就能够提高
  2. Feedback (反馈)
    • 及时反馈
    • 主动型反馈
      • 看别人的代码（GitHub，LeetCode）
    • 被动型反馈
      • code review
      • 写博客，输出

如何刷题

切题四件套

  • Clarification (审题)
  • Possible solution (可能的解决办法)
    • compare(time/spac) (比较多种解决方法之间的差别)
    • optimal (找出最优解)
  • Coding (多写)
  • Test cases
